# piano-sheet
A personal collection of piano sheet music, including song titles and original sources.

## Folder Structure

```
рҹ“Ғ popular/      - Popular sheet music collection
рҹ“Ғ night-piano/  - Shi Jin's Night Piano series
рҹ“„ SOURCES.md    - Centralized source tracking for all sheet music
```

## File Naming Convention

### Popular Sheets (`popular/`)

**Practiced sheets:**
- Format: `[Emotion-Difficulty]~Artist - Title.ext`
- Example: `[жҶӮеӮ·-X]~е‘Ёж·ұ - жӯӨз”ҹжғҹдҪ .pdf`
- The `~` symbol indicates sheets that have been practiced

**Unpracticed sheets:**
- Format: `[Emotion-Difficulty]Artist - Title.ext`
- Example: `[жҶӮеӮ·-X]дёҖж”ҜжҰҙи“® - жө·еә•.pdf`
- No `~` symbol for sheets not yet practiced

### Night Piano Sheets (`night-piano/`)

- Format: `[еӨңзҡ„йӢјзҗҙжӣІX-Difficulty]Number(Title).ext`
- Example: `[еӨңзҡ„йӢјзҗҙжӣІпјҗ-X]1(дёҖеҖӢдәә).pdf`
- No `~` marker as these are series collections

## Emotion Categories

- **е№ійқң** (Calm) - Peaceful, soothing pieces
- **ж„үжӮ…** (Joyful) - Upbeat, cheerful pieces
- **жҶӮеӮ·** (Melancholic) - Sad, touching pieces
- **жҝҖеӢө** (Inspiring) - Uplifting, energetic pieces
- **жҮ·иҲҠ** (Nostalgic) - Memorial, reminiscent pieces

## Difficulty Levels

## Source Management

All sheet music sources are documented in [SOURCES.md](SOURCES.md).

### For Popular Sheets
- Organized by emotion categories (е№ійқң, ж„үжӮ…, жҶӮеӮ·, жҝҖеӢө, жҮ·иҲҠ)
- Each sheet includes:
  - **Source URLs** with labels (e.g., иІјж–Ү, дёӢијү, еҺҹз¶ІеқҖ)
  - **Availability status**: вң… Available / вқҢ Deleted / вҡ пёҸ Moved
  - **Notes** for special conditions

### For Night Piano Series
- Organized by chapters (пјҗ, в… , в…Ў, в…ў, в…Ј, V, в…Ҙ)
- **General sources**: URLs applicable to entire chapters
- **Per-song sources**: Individual URLs for specific songs with their titles
- Special notes marked with italics (e.g., *з„ЎжӣІиӯңдҫҶжәҗ*, *жҡ«з„ЎдҫҶжәҗ*)

### Adding New Sheets
When adding new sheet music:
1. Save the file following the naming convention
2. Add source information to `SOURCES.md` under the appropriate section
3. Include availability status and any relevant notes
