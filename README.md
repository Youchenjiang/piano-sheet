# piano-sheet
A personal collection of piano sheet music, including song titles and original sources.

## 命名規範

所有琴譜檔案遵循以下命名格式：

### 已接受的譜子
格式：`[情緒-難度]~藝術家 - 曲名.副檔名`

範例：
- `[憂傷-X]~周深 - 此生惟你.pdf`
- `[愉悅-X]~周杰倫 - Mojito.txt`

**說明：**
- `~` 符號表示已經練習過、接受的譜子
- 難度使用 `X` 表示尚未評估

### 待嘗試的譜子
格式：`[情緒-難度]藝術家 - 曲名.副檔名`

範例：
- `[憂傷-X]一支榴蓮 - 海底.pdf`
- `[愉悅-X]灰澈 - 星茶會.txt`

**說明：**
- 無 `~` 符號，表示尚未練習
- 難度使用 `X` 表示尚未評估

## 情緒分類

- **平靜** - 寧靜、舒緩的曲風
- **愉悅** - 輕快、歡樂的曲風
- **憂傷** - 悲傷、感人的曲風
- **激勵** - 振奮、激昂的曲風
- **懷舊** - 回憶、思念的曲風

## 難度標記

目前所有譜子的難度標記為 `X`（未評估），後續可以更新為：
- `易` - 簡單
- `中` - 中等
- `難` - 困難

或使用數字等級（如 Lv1-Lv5）

## 資料夾結構

```
📁 已接受/ - 已經練習過、喜歡的譜子
📁 待嘗試/ - 從 media collection 新找的譜子
📁 夜的鋼琴曲/ - 石進的夜的鋼琴曲系列
```
